# Tic Tac Toe AI – CustomTkinter

A modern Tic Tac Toe game built using **Python** and **CustomTkinter**, featuring:
- Dark & Light themes
- Player vs Player mode
- Player vs AI mode
- AI difficulty levels: Easy, Medium, Hard (Minimax)

This project is beginner-friendly and suitable for **college mini-projects** or **AI fundamentals demos**.



# Features

✅ Modern GUI using CustomTkinter  
✅ Dark / Light theme toggle  
✅ Two-player mode (PvP)  
✅ AI opponent mode  
✅ Easy AI – Random moves  
✅ Medium AI – Win & block logic  
✅ Hard AI – Minimax algorithm  
✅ Win highlighting  
✅ Draw detection  
✅ Back to menu option  



# Tech Stack

- Python 3.x
- CustomTkinter
- Minimax Algorithm (AI)
