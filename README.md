# Tic Tac Toe AI – Python GUI with CustomTkinter

A beautiful and interactive **Tic Tac Toe desktop game** built with **Python** and **CustomTkinter** featuring:  
- Two-player mode  
- AI opponent with 3 difficulty levels (Easy, Medium, Hard)  
- Light & Dark themes  
- Unbeatable AI using Minimax

Perfect for learning GUI development, game logic, and basic AI algorithms in Python.

## How to Play

1. Select **theme** (Light / Dark)
2. Choose **Play Mode**
    - Two Players (PvP)
    - Play vs AI
3. If AI, choose **difficulty**
    - **Easy** – Random AI moves
    - **Medium** – Blocks & basic strategy
    - **Hard** – Minimax (optimal play)
4. Place your move by clicking on an empty cell
5. First player to align 3 symbols wins
6. Winning combination is highlighted

Press **Back to Menu** to start again.

## Game Rules

- Player **X** always starts
- Players take alternating turns
- Game ends when:
    - A player wins
    - All spots are filled (draw)
